User-facing

/dashboard
Updated: hero card with active program + "Continue" CTA. Program progress metric card.
/programs
Grid of programs matching user's category + plan. Shows progress % if enrolled. Locked programs show upgrade CTA.
/programs/[id]
Program detail: cover, description, week list. "Start Program" enrolls the user. "Continue Week X" if already enrolled.
/programs/[id]/week/[n]
Week detail: sessions list with complete/incomplete state.
/programs/[id]/week/[n]/session/[s]
Session view: exercises with sets/reps, auto-calculated weight from user's 1RM. Completion checkbox per exercise. "Mark session complete" CTA writes to user_session_completions.
/exercises
Exercise library. Filter by muscle group + tags. Card grid.
/exercises/[id] NEW
Exercise detail: name, muscle groups, instructions, sets/reps context, video placeholder (SP-6 fills in). Was missing from previous version.
/settings UPDATED
New fields from brief §11:
• Max lifts card (squat/bench/deadlift/OHP in kg)
• Phone number field
• Avatar upload
• Name + email editing (brief §11 — currently read-only, closing the gap now since we're here)
