SP-4 Routes

Every new page being built. User-facing on the left, coach admin on the right.

User-facing

/dashboard
Updated: hero card shows active program + "Continue" CTA. New program progress metric card.
/programs
Grid of programs matching user's category + plan. Shows progress % if enrolled. Locked programs show upgrade CTA.
/programs/[id]
Program detail: cover, description, week list with lock state. "Start" or "Continue Week X" CTA.
/programs/[id]/week/[n]
Week detail: title, description, session list. Each session shows exercises preview + complete/incomplete state.
/programs/[id]/week/[n]/session/[s]
Session view: exercise list with sets/reps, auto-calculated weight from 1RM %, completion checkbox per exercise. "Mark session complete" CTA.
/exercises
Exercise library. Filter by muscle group + tags. Each card shows name, muscle groups. Tap to see detail + instructions.
/settings
Updated: new Max Lifts card where user enters squat/bench/deadlift/OHP in kg.

Coach / Admin

/admin
Admin dashboard/index. Links to exercises and programs. Role-gated: profiles.role = 'admin'.
/admin/exercises
Table of all exercises. Create, edit, delete. Filter by muscle group/tags.
/admin/exercises/new
/admin/exercises/[id]/edit
Form: title, description, muscle groups (multi-select), tags, optional video URL.
/admin/programs
Table of all programs. Create, edit, delete. Filter by category + status. Toggle active/draft.
/admin/programs/new
Form: title, description, cover image URL, category, plan access, status. Creates the program shell — weeks/sessions added in edit view.
/admin/programs/[id]/edit
The main builder: add/edit/delete weeks → sessions → exercises. Each session_exercise has sets/reps/weight/notes. Plain form-based (SP-7 adds drag-and-drop).
