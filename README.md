# Elevate Coaching

Premium training platform for fitness coach. Three-tier subscription (Free / Basic £150 / Pro £300) with category-based content publishing for the coach to serve 400+ users.

## Status

Currently building SP-1 (Platform Spine) — the vertical slice of the 8-sub-project sequence.

- Design: `docs/superpowers/specs/2026-05-13-sp1-platform-spine-design.md`
- Plan: `docs/superpowers/plans/2026-05-13-sp1-platform-spine.md`

## Stack

Next.js 16 (App Router) + Neon Postgres (RLS) + Clerk + Stripe + Cloudflare Stream + Vercel Blob + shadcn/ui.

---

## Getting Started

Install deps and run the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:3000.
